# 第1章 总体架构

## 1.1 设计理念

jQuery 是一款革命性的 JavaScript 库，秉承着 "以用为本" 的设计理念，倡导 "写更少的代码，做更多的事情" (write less, do more)，极大地
提升了 JavaScript 开发体验。

jQuery 的核心特性可以总结为：

- 兼容主流浏览器，支持 IE 6.0+、Chrome、Firefox 3.6+、Safari 5.0+、Opera 等。
- 具有独特的链式语法和短小清晰的多功能接口。
- 具有高效灵活的 CSS 选择器，并且可对 CSS 选择器进行扩展。
- 拥有便捷的插件扩展机制和丰富的插件。
